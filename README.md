# CPU Scheduling Simulator

This project is based on Operating System concepts.  
It includes implementation of basic CPU scheduling algorithms.

## Algorithms Covered
- FCFS (First Come First Serve)
- SJF (Shortest Job First)
- Round Robin

## What this project does
- Takes process burst time as input
- Calculates Waiting Time (WT)
- Calculates Turnaround Time (TAT)
- Displays results in table format

## Language Used
C Programming
